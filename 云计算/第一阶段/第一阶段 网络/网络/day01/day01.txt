网络拓扑结构
1、点对点
两台设备之间有一条单独的连接，通常用于广域网连接

2、星型拓扑
1）优点：
         1.易于实现       #交换机接入pc机即可   
         2.易于网络扩展  #如果交换机的接口不够再利用交换机的一个接口用根网线再连一台交换机即可
         3.易于故障排查  #理论来说每个电脑上的接口都有标签


2）缺点：中心节点压力大、组网成本较高

3、网型拓扑结构
1）各个节点至少与其他两个节点相连
2）可靠性高、组网成本高



二、网络通信参考模型
1、OSI参考模型
1、国际标准化组织（ISO）
开放系统互连参考模型OSI
OSI是一个开放式体系结构，它规定将网络分为七层

2、协议分层
为了降低网络设计的复杂性，将协议进行了分层设计

应用层：网络服务与最终用户的一个接口
表示层：数据的表现形式，如加密、压缩。
会话层：建立、管理、中止会话，例如断点续传。
传输层：定义传输数据的协议端口号，以及流控和差错校验。
网络层：进行逻辑地址寻址，实现不同网络之间的通信。
数据链路层：建立逻辑连接、进行硬件地址寻址、差错校验等功能。
物理层：建立、维护、断开物理连接。


2、TCP/IP协议族的组成
应用层：HTTP、https、FTP、TFTP、SMTP 、SNMP、DNS
传输层：TCP、UDP
网络层：ICMP（ping是工具，协议是种定义了某种解决方案，用工具去实现解决方案，工具可以有很多种）、IGMP、IP、ARP
数据链路层、物理层：由底层网络定义的协议


3、PDU（协议数据单元）
应用层                                                                                                       #计算机
传输层     段    segment  |TCP头部|          上层数据                                                    #防火墙
网络层     包    packet   |IP头部|TCP头部|  上层数据                                                     #路由器
数据链路层 帧    frame    |MAC头部|IP头部|TCP头部|  上层数据                                            #交换机
物理层     比特  bit（传输数据的单位，二进制）  #储存单位 100M=100 000KB= 100 000 000B  字节）    #网卡

三、物理层
物理层-网络的基础   #物理层是网络的基础,正如同公路是汽车通行的基础一样
1.物理层是TCP/IP模型的最底层
2.为数据端设备提供传送数据的通路

1.接口
以太网接口：
RJ-45水晶头
光纤接口：
FC 、ST、SC 、LC 窄体方形光纤接头（目前主流）、MT-RJ 

2.双绞线
1）双绞线分类：
屏蔽双绞线 （STP）
线外包裹一层金属网膜，用于电磁环境非常复杂的工业环境中 
非屏蔽双绞线 （UTP）
2）双绞线标准与分类 ：
Cat 5五类双绞线，适用于100Mbps的网络
Cat 5e衰减更小，适用于100Mbps的网络，串扰更少，性能优于5类线 （超五类）
Cat 6适用于1000Mbps的网络
Cat 7适用于10000Mbps的网络
3)双绞线的连接规范
1）线序
T568A：
白绿、绿、白橙、蓝、白蓝、橙、白棕、棕
T568B：
白橙、橙、白绿、蓝、白蓝、绿、白棕、棕 
2）线缆的连接：
标准网线（直连线或直通线）：用于连接不同设备（A-A，B-B）
交叉网线：用于连接相同设备 （A-B）
全反线 ：不用于以太网的连接，主要用于计算机的串口和路由器或交换机的console（控制口）相连
例外情况：版本较新设备可以随意使用标准与交叉网线而不受限制，设备本身具备自动识别功能。

四、配置交换机
1.交换机的工作模式：
1、Switch>                                     #用户模式

2、Switch>enable                               #Switch#特权模式

3、Switch#configure  terminal                  #Switch(config)#全局配置模式

4、Switch(config)#interface fastEthernet 0/1   #Switch(config-if)#接口模式

5、exit返回上一模式

6、end直接退到特权模式 #ctrl+Z

命令输入错误被卡住时同时按键盘ctrl+shift+6这三个按键  #实验时

2、常用命令： 
1.Switch(config)#hostname S1    #修改主机名为S1

2.Switch#show  running-config   #查看配置信息

3.配置enable明文口令                   #配置进入特权模式的密码
全局配置模式：enable  password  123

保存交换机的配置
特权： copy  running-config  startup-config 或 write

恢复设备出厂默认值
特权：erase  startup-config   #擦除配置文件
重启：reload

4.设备配置的准备工作                #防止有人去查看交换机
空闲一段时间后，重回初始界面的问题
switch(config)#line con 0
switch(config-line)#exec-timeout 0 0
 
5.配置输出日志同步                  #工作中会有机器的开机关机，可以不让弹出的信息打断配置过程
Switch(config)#line console 0
Switch(config-line)#logging synchronous

6.禁用DNS查询#                     #打错命令，交换机会以为是域名，会反应一段时间，禁用之后就不会再去查询
switch(config)#no ip domain-lookup
